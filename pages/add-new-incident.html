Add New Incident

Description

This section will guide the admin through adding a new Incident on the IMOS System.

Prerequisites

Admin must be logged into the system

Adding a new Incident process

Step 1: Select the Incident Button on the Incident Card on the home page.

Step 2: Select the Add button placed on the top right of the screen.

Step 3: Enter information in all the empty text fields, all fields need to be populated with the correct information relating to the admin item being added.

Â Step 4: Click on the Add button to proceed with the Incident addition or the Cancel button to stop the process.

Step 5: Click on the OK button on the confirmation notification to confirm the addition or the cancel button to return to the add Incident screen

Exercise

Click the Add button (top left) on the Incident detail screen.

Submit a new Incident detail with your own details:

Click the save button to add the Incident.

Click the Cancel button to return to Incident detail screen.

Summary

You have now learnt how to add a new Incident successfully with its corresponding details.

In order to successfully add a new Incident all, the required fields need to be provided in the correct format.
