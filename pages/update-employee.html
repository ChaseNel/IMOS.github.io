Update Employee

Description

This section will guide the admin through Updating an existing Employee on the IMOS System.

Prerequisites

Admin must be logged into the system

Updating a Employee process

Step 1: Select the Employee Button on the Employee Card on the home page.

Step 2: Select the Edit button that is next to the employee you want to update..

Step 3: Enter the changed information the necessary text fields, all fields need to be populated with the correct information relating to the admin item being updated.

Step 4: Click on the Update button to proceed with the Employee being updated or the back button to stop the process.

Step 5: Click on the OK button on the confirmation notification to confirm the update or the cancel button to return to the add Employee screen.

Exercise

Click the Edit button (yellow button) on the Employee detail screen.

Submit a updated Employee detail with your own details:

Click the update button to update the Employee.

Click the Cancel button to return to Employee detail screen.

Summary

You have now learnt how to update a new Employee successfully with its corresponding details.

In order to successfully update a new Employee all, the required fields need to be provided in the correct format.
