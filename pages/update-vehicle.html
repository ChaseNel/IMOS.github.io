Update Vehicle

Description

This section will guide the admin through Updating an existing Vehicle on the IMOS System.

Prerequisites

Admin must be logged into the system

Updating a Vehicle process

Step 1: Select the Vehicle Button on the Vehicle Card on the home page.

Step 2: Select the Edit button that is next to the Vehicle you want to update.

Step 3: Enter the changed information the necessary text fields, all fields need to be populated with the correct information relating to the admin item being updated.

Step 4: Click on the Update button to proceed with the Vehicle being updated or the back button to stop the process.

Step 5: Click on the OK button on the confirmation notification to confirm the update or the cancel button to return to the add Vehicle screen.

Exercise

Click the Edit button (yellow button) on the Vehicle detail screen.

Submit a updated Vehicle detail with your own details:

Click the update button to update the Vehicle.

Click the Cancel button to return to Vehicle detail screen.

Summary

You have now learnt how to update a new Vehicle successfully with its corresponding details.

In order to successfully update a new Vehicle all, the required fields need to be provided in the correct format.
