Generate Supplier Order

Description

This section will guide the admin through adding a Generate Supplier Order on the IMOS System.

Prerequisites

Adding a new Generate Supplier Order process

Step 1: Select the Supplier Button on the Supplier Card on the home page.

Step 2 Select the Generate Supplier Order Burron on the Supplier Page

Step 3: The Generate Supplier Order will be generated.
