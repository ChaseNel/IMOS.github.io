Help > Reports Instructions > Generate Material Request Report > Steps
Steps

Step 1: Select the Generate Material Request Report Button on the report Card on the home page.

Step 2: Select the Generate Report button in the Material Request detail table on the report detail screen.

Step 3: Click on the OK button on the confirmation notification to confirm the Generate Material Request Report or the cancel button to return to the Report detail screen.
