Search and Delete Task

Description

This section will guide the admin through searching for a Task and deleting one on the IMOS System.

Prerequisites

Admin must be logged into the system

Task must exist on the system

Searching of a Task process

Step 1: Select the Task Button on the Task Card on the home page.

Step 2: Click on the Search bar on the Task (top left) detail screen.

Step 3: Input the search criteria.

Exercise

Click on the Search bar on the Task detail screen.

Type in the search criteria based on any of the fields.

Summary

You have now learnt how to search an Task successfully.

The search function searches for any record that matches the search criteria within Task table.

Delete a Task process

Step 1: Select the Task Button on the Task Card on the home page.

Step 2: Select the delete button in the Task detail table on the Task detail screen.

Step 3: Click on the OK button on the confirmation notification to confirm the deleting or the cancel button to return to the Task detail screen.

Exercise

Click the delete button in the Task detail table on the Task detail screen.

Click the yes button to delete the Task.

Click the Cancel button to return to Task detail screen.

Summary

You have now learnt how to delete an Task successfully.
